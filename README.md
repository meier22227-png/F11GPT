# ⚽️ Football Tactician & Training Assistant Telegram Bot

A lightweight, self-contained, and fully compliant Telegram bot built with Node.js and the Telegraf framework. This application serves as an interactive, offline educational playbook and utility manual for amateur football players, grassroots coaches, and sports enthusiasts.

## 🚀 Features

- **📐 Tactical Playbook:** Detailed breakdowns of classic football formations (4-4-2, 4-3-3, 3-5-2) detailing tactical strengths, weaknesses, and key positional responsibilities.
- **🏃‍♂️ Interactive Training Drills:** Step-by-step setup guides and coaching focal points for essential skill circuits (e.g., Rondos, Clinical Shooting).
- **💪 Fitness & Conditioning Programs:** Tailored High-Intensity Interval Training (HIIT) and agility ladder routines optimized for football athletes.
- **❓ Policy Compliant About Section:** Clear documentation explicitly proving compliance with platform rules.

## 🛑 Policy & Compliance Standards

This bot has been architected from the ground up to strictly adhere to the **Telegram Ad Platform Policies**:
- **Zero Trademark Infringements:** No use of copyrighted club names, league names, or protected tournament branding.
- **No Gambling/Betting:** Contains absolutely no match predictions, betting tips, odds analysis, or financial gambling links.
- **No Copyright Violations:** No live stream links, copyrighted match video highlights, or pirated media streams.
- **Purely Educational:** Serves strictly as a static data utility to aid amateur athletic development.

## 🛠 Tech Stack & Deployment

- **Runtime:** Node.js (>= 18.0.0)
- **Framework:** Telegraf (v4)
- **Deployment Platform:** Hosted seamlessly on Railway via GitHub integration.
